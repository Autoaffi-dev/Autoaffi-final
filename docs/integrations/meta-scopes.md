# Meta API Scopes used by Autoaffi

Autoaffi uses the following scopes:
- `pages_read_engagement`
- `pages_show_list`
- `business_management`
- `ads_read`
- `instagram_basic`
- `instagram_manage_insights`

✅ Only minimal scopes needed for reading engagement data and ads performance. 
❌ No posting or DM access without explicit user consent.
